# MythOS: Mythological Operating System ⚡🔮

MythOS는 고대 신화의 지혜를 현대적인 인터페이스로 재해석한 플랫폼입니다. 신탁(Oracle) 시스템을 통해 사용자의 고민에 대한 신화적 통찰력을 제공하며, 인류 공통의 심리 패턴인 '원형(Archetypes)'과 시대를 초월한 '지혜(Wisdom)'를 탐구할 수 있는 디지털 환경을 제공합니다.

## ✨ 주요 특징

- **🔮 The Oracle:** 사용자의 질문에 대해 신화적 참조와 가이드를 실시간으로 제공.
- **⚔️ Archetypal Library:** 융(Jung)의 원형 이론을 바탕으로 한 캐릭터 패턴 분석.
- **📚 Ancient Wisdom:** 전 세계 다양한 문화권(그리스, 이집트, 북유럽 등)의 신화 데이터베이스.
- **🌑 Dark Mode OS UI:** 프리미엄 블랙 테마와 글래스모피즘(Glassmorphism) 기반의 유려한 디자인.
- **🏗️ Modern Tech Stack:** React, Vite, Node.js, Express, Tailwind CSS 사용.

## 🚀 시작하기

### 📦 필수 설치 요소
- [Node.js](https://nodejs.org/) (v16.x 이상 권장)
- [npm](https://www.npmjs.com/)

### 🛠️ 설치 및 로컬 실행

1. **저장소 클론**
   ```bash
   git clone https://github.com/doryeng/my-mythos.git
   cd my-mythos
   ```

2. **백엔드 서버 실행**
   ```bash
   cd backend
   npm install
   npm start
   ```

3. **프론트엔드 실행**
   ```bash
   cd ../frontend
   npm install
   npm run dev
   ```

## 🛠️ 기술 스택 (Tech Stack)

- **Frontend:** React, Vite, Tailwind CSS, Lucide React (Icons)
- **Backend:** Node.js, Express, Cross-Origin Resource Sharing (CORS)
- **Deployment:** GitHub Pages / Vercel (추천)
